How splitting works (in plain English)

A PDF is a stack of pages. When you load your file, our tool reads the page count and waits for your instructions. Your ranges tell us which page numbers you want to keep. On split, we copy those page objects into a new, empty document in the same order you requested. The original file stays exactly the same.

Because this is a page-copy operation (not screenshots), your text, links, vector graphics, and image quality are preserved. Mixed orientations and sizes are fine—portrait pages can sit next to landscape slides without any stretching.

What it is (and isn’t)

Splitting keeps pages; it doesn’t edit content inside a page. If you need to remove pages from the original, use Delete Pages. If you want to squeeze the new file smaller for email, run it through the Compressor. To combine multiple extracts, use the Merge tool afterward.

Privacy by default

Everything happens on your device. We don’t upload, scan, or store your documents, which makes the tool safe for contracts, IDs, grades, and confidential reports. Close the tab, and the working data disappears with it.

FAQs

Will quality change after splitting?

No. We copy page objects directly; text, links, and graphics remain intact.

Can I split a very large PDF?

Yes, but memory depends on your device. If it feels slow, split in chunks (e.g., chapters) and merge later if needed.

Can I rearrange while splitting?

Splitting follows numeric order. If you need a different sequence, export the subset then use the Reorder tool.
