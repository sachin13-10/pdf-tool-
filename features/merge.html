Merging Tips

Merging PDFs is straightforward, but a few smart moves will save you from having to redo the whole thing. Here's what you need to know:

  • File Order Matters: PDFs are merged in the exact order you select them. If you accidentally pick files out of sequence, don't worrymost file browsers let you Ctrl+Click (or Cmd+Click on Mac) to select multiple files in the order you want. Just be intentional about it from the start.
  • Add More Files Anytime: You can keep selecting and adding PDFs even after the first batch loads. This is super handy when you're working with files scattered across different folders‚Äîjust grab a few at a time and build your merged document piece by piece.
  • Mixed Sizes Are Totally Fine: Don't worry about mixing landscape and portrait pages, or combining A4 with Letter-sized PDFs. The tool preserves each page's original size and orientation, so everything stays intact. The result might look a bit inconsistent when flipping through, but it's perfectly valid.
  • Large Merges Need Memory: If you're combining massive PDFs (100+ pages each), close other browser tabs and applications to free up RAM. PDF processing is memory-intensive, and running out of space mid-merge can crash the browser or make things painfully slow.
  • Name It Clearly Right Away: When you download the merged PDF, use a descriptive filename like Q1-2025-Reports-Merged.pdf or Client_Proposal_Complete.pdf. Future you will thank you when searching for this document six months from now.
  • Check Before You Share: Always open the merged PDF and flip through it before sending it off. Make sure pages are in the right order, nothing got skipped, and the file size is reasonable. A quick 30-second check prevents embarrassing mistakes later.

When to Merge vs. Split

Not sure if you should be merging or splitting? Here's a quick decision guide:

Use Merge when: You have separate PDF files that belong together‚Äîlike invoice copies, contract sections, or report chapters. Merging creates one convenient file that's easier to share, store, and organize. It's perfect for bundling related documents into a single package.

Use Split when: You have one large PDF but only need certain pages or sections. For example, if you received a 50-page contract but only want pages 10-15, use our Split PDF tool to extract just what you need. Splitting also helps when a file is too large to email‚Äîbreak it into smaller chunks first.

Pro tip: If you're merging multiple large PDFs and the result is too big to email (most email servers cap attachments at 25MB), consider using our Compress PDF tool afterward to shrink the file size without losing much quality.

Frequently Asked Questions

Do page sizes change when merging?

No. Each page keeps its original size and orientation. You can merge PDFs with different page sizes (A4, Letter, landscape, etc.) and they'll all coexist in the merged file.

Can I control the order of PDFs?

Yes! PDFs are merged in the exact order you select them. If you need a different order, select the files in the sequence you want them to appear in the final document.

Will links and formatting be preserved?

Absolutely! We copy pages as-is, preserving all embedded text, images, links, fonts, and vector graphics. Nothing is re-rendered or convertedit's a direct page copy operation.

What if one file is password-protected?

Password-protected PDFs will be skipped during the merge. You'll need to unlock them first using our Unlock tool, or remove them from your selection and merge the rest.

Is there a limit on file size or page count?

The tool can handle PDFs of any size, but very large files (100+ MB or 500+ pages) may take longer depending on your browser's memory. For massive merges, try combining in smaller batches first.

Can I add files from different folders?

Yes! You can select files from any location on your computer. Make multiple file selections from different folders, and they'll all be added to the merge queue.
